Программа представляет собой инструмент командной строки для преобразования конфигурационных файлов, написанных на учебном конфигурационном языке, в формат TOML. Инструмент выполняет синтаксический разбор входного файла, проверяет его корректность и генерирует эквивалентный TOML-файл.
Числа: -?(\d+|\d+\.\d*|\.\d+)([eE][-+]?\d+)?

Примеры: 42, 3.14, -2.5, 1.2e-3

Строки: q(текст)

Примеры: q(Hello World), q(), q(123)

Словари: {имя = значение}
